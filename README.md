# Citation Badge

## Usage

```
https://citations.njzjz.win/<DOI>
```

For example, insert the following line in markdown:

```markdown
![Citations](https://citations.njzjz.win/10.1038/s41467-020-19497-z)
```

The following badge will be shown:
![Citations](https://citations.njzjz.win/10.1038/s41467-020-19497-z)
